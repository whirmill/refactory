# Memory - 2026-04-13

## Refactory Project Setup

- **Repository:** git@github.com:whirmill/refactory.git (public)
- **Live site:** https://whirmill.github.io/refactory/
- **Tech stack:** Vite + React 19 + TypeScript + Tailwind CSS v4 + shadcn/ui
- **Content:** Blog di storie umoristiche sulla vita da sviluppatore (in italiano)

## Cronjobs Configured

1. **refractory-story** - Genera nuova storia ogni 10 minuti
   - Session: isolated
   - Prompt: Legge storie esistenti, genera contenuto originale, commit+push
   - Git config required in prompt (user.name, user.email)

2. **stop-refractory-stories** - Ferma generazione alle 08:00 UTC del 2026-04-14
   - One-shot, auto-deletes after run

## GitHub Pages Setup

- Router: `HashRouter` (BrowserRouter non funziona su GitHub Pages)
- Base path: `/refactory/` in vite.config.ts
- SPA routing: 404.html redirect trick
- Deploy: GitHub Actions workflow (.github/workflows/deploy.yml)

## Privacy Notes

- Workspace files (AGENTS.md, SOUL.md, USER.md, etc.) in .gitignore
- History squashed to single commit to remove workspace files from git history
- Repo made public via `gh repo edit --visibility public`

## Key Learnings

- Cron isolated sessions need explicit `cd` and git config in prompt
- `import.meta.glob` bundles markdown files into JS at build time
- GitHub Pages requires HashRouter for SPAs without server config
